Netflix观影指南

2021-05-27 10 min read

1. Netflix是什么？

Netflix 是一家优质的视频服务商，与国内腾讯视频、爱奇艺类似。但与之不同的是，Netflix 并无免费内容，所有在 Netflix 上的电影、剧集等，都需要付费购买会员才可以观看。并且 Netflix 自制剧（类似爱奇艺自制的网剧）几乎都是神剧，评分爆炸。目前越来越多的用户开始接触并选择观看 Netflix 。

2. 大陆地区如何观看 Netflix？

大陆地区用户想要观看 Netflix 有两个问题需要解决：

2.1 会员账号及价格

Netflix 账号在使用时，可以当前 IP 所在地的片库，并没有锁定付费地区片库。也就是我们可以选择会员费低的地区付费，然后用片库更丰富的地区 IP 进行观看。当然这样做也有一定风险，有可能 Netflix 会封禁你的账号。但是目前来看，用此方法是可行的，没有出现大规模封禁的事件（土耳其、阿根廷区我会在下面详细说明）。

2.1.1 用户限制说明

每一个 Netflix 账号最多只能建立 5 个独立的 Profile ，如果你不想与他人的观看记录混在一起的话，最好一个账号的使用者不超过 5 人，每个人使用自己独立的 Profile 。

2.1.2 会员等级说明

首先，Netflix针对全球不同地区的定价也是不同的，而会员又分为三个等级，Basic、Standard、Premium。其中某些地区有使用当地手机扣费的 Mobile 等级，和支持 HDR 的 Ultra 等级，不在我们讨论范围之中。

Basic 是性价比最低的等级，价格一般有Premium 的50%甚至更多，但是只限制1台设备同时播放，并且最高只能观看 480p 分辨率的影片。
Standard 是性价比稍高的等级，价格一般有Premium 的70%-80%，限制2台设备同时播放，可以观看1080p分辨率的影片，适合和家人、熟悉的朋友，并且主要在手机、iPad这种移动设备观影使用。
Premium 是性价比最高的等级，限制4台设备同时播放，可以观看4K分辨率的影片，适合有4K电视的用户使用，或者想节省成本，与他人公用账号。

同时观看设备数 支持分辨率
Basic 1 480p
Standard 2 1080p
Premium 4 4K

2.1.3 最经济的Netflix付费地区

目前比较经济的地区为土耳其、巴西、阿根廷。
其中，土耳其区早期价格十分低廉，被多数用户利用，现在已经涨价，并且封禁了香港的 IP，但是价格仍然十分诱人。
阿根廷区的价格是不含税的价格，同样的被多数用户利用，现在在阿根廷区消费 Netflix 会员需要额外的64%税费，即便有高昂的税费，价格在全球看来还是比较低的。（随着汇率的变动，阿根廷区的价格涨了20%，但是在全球看来，仍然是低价区）

2.2 地区限制

这个问题比较麻烦，Netflix会限制商用IP，不像买个国外服务器开代理就能上谷歌推特一样简单，所以你在服务商购买的服务器很大概率无法观看Netflix，有小部分可以观看 Netflix 的服务器但是价格也会比较高。所以购买一家 可看 Netflix 的机场 服务是绝大多数用户的选择。
懂一点服务器知识的朋友可以自建代理学习解锁 Netflix，让无法观看 Netflix 的 IP 顺利解除观看限制，可以看看我的另一篇博文

3. 观看设备

3.1 在 Windows 上观看 Netflix

在 Windows 上可以使用 Netflix Win10 应用版 / 浏览器 观看 Netflix 。

Netflix Win10 应用版，最高支持 4K 分辨率。在 Windows 商店搜索 Netflix 即可安装，如果搜索不到，很可能因为你的 商店是中国区，请参考 微软官方教程 进行换区。 因为一般的 Shadowsocks 代理是无法为 Windows 10 UWP 应用进行代理，如有需要请使用 Clash 或者参考 少数派的教程 进行配置。

Windows 平台各浏览器均可看 Netflix，Edge 浏览器支持最高 4K 分辨率播放。播放 4K 同时对系统、处理器、图形处理器、接口（外接显示屏）都有要求，请参考 Netflix 官方帮助。

Chrome 浏览器在 Windows 播放 Netflix 最高只支持 720P，很多人习惯使用 Chrome，可以安装 Super Netflix 插件后，使用快捷键 Ctrl + Shift + Alt + S 强制更换分辨率至最高 1080P。未安装插件时，使用快捷键也无法切换到 1080P。

使用 Netflix 时的部分浏览器快捷键
Ctrl + Shift + Alt + T 可以添加本地字幕文件
Ctrl + Shift + Alt + D 可以查看目前分辨率情况
Ctrl + Shift + Alt + S 可以强制更换分辨率；更换分辨率时，左侧是声音的码率，中间是视频的码率，右侧是 CDN 的速度，均选择最大值即可。

3.2 在 macOS 设备上观看 Netflix

macOS 上各浏览器均可观看 Netflix。macOS 下 Safari 浏览器满足以下条件后可支持播放 4K 视频，其余情况仅支持最高 1080P。

硬件是 2018年后发布的含有 T2 芯片的新产品；
系统是 macOS Big Sur 及以上；
浏览器版本是 Safari 14 以及上；

Chrome 浏览器在 macOS 上播放 Netflix 最高只支持 720P，很多人习惯使用 Chrome，可以安装 Super Netflix 插件后，使用快捷键 Ctrl + Shift + Alt + S 强制更换分辨率至最高 1080P。未安装插件时，使用快捷键也无法切换到 1080P。

3.3 在 Android 设备上观看 Netflix

说到 Android 设备，首先需要了解绝大部份的国产 Android 设备，无法播放高清 Netflix 视频，最高只能播放 480P 影片。国内销售的国产手机几乎都没有 DRM 认证。DRM 认证即 Digital Rights Management 数字版权管理，不深入解释，此处是指，DRM 认证的设备可以播放高清的视频；而未认证的设备，即使硬件上支持，也无法播放高清，防止高清资源被录制盗版出去。这是国外几大厂商为了保护版权所达成的一种认证，DRM 认证这种做法褒贬不一，但在国外已经成为一种规范。

Netflix 在未进行 DRM 认证的设备上无法提供高清播放。Netflix 官方帮助教程列出了支持 HD 以及 HDR 的 Android 设备。如果你是 Android 设备，需要先查看自己机型的情况，如果没有记录，大概率无法播放高清视频。

如果 Android 设备在 Google Play 上无法搜到 Netflix，请切换 Play 账号到非中国区。换区后如果仍然无法找到，很可能是因为没有 DRM 认证或者 DRM 等级低，Google Play 自动不显示。请尝试在 APKMirror / APK pure 下载 Netflix 的客户端。

如果你的设备支持 Netflix 高清播放，但仍然不够清晰。请检查网络是否稳定，同时需要在 账户设置 中切换到最高分辨率。Netflix 在 Android 上会根据网络情况和用户设置提供合适的分辨率，不能手动切换。

3.4 在 iOS 设备上观看 Netflix

iOS 上如果在 App Store 搜索不到 Netflix，请使用其他非中国区的账号登录 App Store 下载。因为 Netflix 未在中国运营，所以也没有上架中国区 App Store。

iOS 设备十分省心，安装客户端后，配合合适的代理即可使用。所有 iOS 设备都有最高规格的 DRM 认证，均能播放 Netflix 高清。但实际上 Netflix 对不同版本的 iPhone/iPad 支持的最高分辨率并不一样。较新款带有 OLED 屏幕的 iPhone 支持的最高流媒体播放分辨率为 1080p，而 XR 等稍低端或者更旧的设备最高支持 720p。一般情况下，iPad 支持的最高流媒体播放分辨率为 1080p。

3.5 在 Apple TV 上观看 Netflix

Apple TV 不支持代理，在 Apple TV 上使用 Netflix 需要在路由器上配置科学上网环境，或者使用 Surge 等特殊软件进行网关代理。

3.6 在 Android TV 等电视盒子上观看 Netflix

在播放 Netflix 的问题上，电视盒子的标准比上面的 DRM 认证更为严格。设备必须经过 Netflix 认证才能正常安装和播放 Netflix。 在中国主流使用的设备有 Chromecast 系列， 小米盒子国际版， Shield TV 国际版 。

小米盒子国际版

因为没有 Netflix ESN 码，破解 / 刷国际版固件后的小米盒子国内版依旧无法播放 Netflix，这可能也是小米盒子国际版硬件与国内差不多，但是价格要高不少的一个原因。

小米盒子国际版首次开机需要登陆谷歌账号，且此时无法修改 Wi-Fi 代理，这就意味着，首次开机需要科学上网环境。开机之后，可安装已对 Android TV 进行优化的 ClashForAndroid 最新版进行代理。

Shield TV

京东上购入的国行 Shield TV 含有 ESN 码，直接安装 Netflix TV 版即可即可使用，无需刷机。APK 安装包可以前往 APK Mirror 下载）
